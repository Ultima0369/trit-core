## 一、为什么是小波？

### 1.1 傅里叶变换的问题

**假设信号是平稳的**：频率不随时间变化。

但人的注意力是**非平稳**的：
- 基频会漂移（从早睡早起变成晚睡晚起）
- 谐波会重构（周循环消失，项目周期成为主导）
- 新频率会出现（新压力源产生新节律）

**傅里叶变换告诉你"有哪些频率"，但不告诉你"这些频率什么时候出现"。**

### 1.2 阈值检测的问题

**假设基线是稳定的**："正常 = 历史平均值 ± 20%"

但人的"正常"本身在变化：
- 季节性：冬季社交频率低，夏季高
- 生命阶段：创业期工作强度高，稳定期降低
- 环境适应：新城市的新基线可能永久不同于旧城市

**阈值系统产生大量假阳性（"他异常了"但其实只是基线漂移了）和假阴性（"他正常"但谐波结构已经改变）。**

### 1.3 小波变换的优势

- **同时在时间域和频率域分析**：知道"什么频率在什么时候出现"
- **检测瞬时变化**：某一天的异常 + 长期漂移
- **识别多尺度模式**：分钟级情绪波动 + 周级压力周期 + 年级人生阶段
- **自适应窗口**：高频信号用窄窗口，低频信号用宽窗口

## 二、连续小波变换（CWT）

### 2.1 定义

$$W_x(a, b) = \frac{1}{\sqrt{a}} \int_{-\infty}^{\infty} x(t) \cdot \psi^*\left(\frac{t-b}{a}\right) dt$$

**参数说明**：
- $x(t)$：输入信号（连续时间序列）
- $\psi$：母小波（Morlet 小波，适合振荡信号）
- $a$：尺度（对应频率的反比，$a > 0$）
- $b$：时间平移（$b \in \mathbb{R}$）
- $\psi^*$：母小波的复共轭

**物理意义**：
- 在时间点 $b$、频率尺度 $a$ 上，信号 $x(t)$ 与母小波的相似度
- $a$ 大 → 低频 → 宽窗口（看长期趋势）
- $a$ 小 → 高频 → 窄窗口（看瞬时变化）

### 2.2 Morlet 母小波

$$\psi(t) = \pi^{-1/4} e^{i\omega_0 t} e^{-t^2/2}$$

**参数说明**：
- $\omega_0$：中心频率（典型值 6.0，平衡时间分辨率和频率分辨率）
- 复指数 $e^{i\omega_0 t}$：振荡成分
- 高斯包络 $e^{-t^2/2}$：局部化成分

**为什么选择 Morlet**：
- 在频率域有良好的局部化（高斯形状）
- 在时间域也有良好的局部化（高斯包络）
- 平衡了时间分辨率和频率分辨率的矛盾（Heisenberg 不确定性）

### 2.3 小波功率谱

$$P(a, b) = |W_x(a, b)|^2$$

**物理意义**：
- 在时间点 $b$、频率尺度 $a$ 上，信号的能量密度
- 高功率 = 在该时间-频率点上，信号有强周期性成分

## 三、认知特征提取

### 3.1 基频（Fundamental Frequency）

$$f_0 = \arg\max_f \left\{ \frac{1}{T} \int_0^T P(f, t) dt \right\}$$

**物理意义**：
- 一个人的"日常节奏"——主频率
- 日型人：$f_0 \approx 1/24h$（昼夜节律）
- 夜型人：$f_0 \approx 1/24h$ 但相位滞后
- 创业者：$f_0$ 可能偏向 $1/7d$（周循环）

**映射到 Trit-Core `Phase`**：
- 基频稳定性 → 系统稳定性的代理指标
- 基频漂移 → 生活节奏改变的信号

### 3.2 谐波（Harmonics）

$$f_k = k \cdot f_0, \quad k = 2, 3, 4, ...$$

**认知意义的谐波**：

| 谐波次数 | 频率 | 对应节律 | 检测方式 |
|----------|------|----------|----------|
| $k=1$ | $f_0$ | 基频（昼夜/周/月） | 主频率峰值 |
| $k=2$ | $2f_0$ | 半周期（半天/半周） | 功率谱次峰值 |
| $k=7$ | $7f_0$ | 若 $f_0=1/d$，则 $7f_0=1/7d$ | 周循环检测 |
| $k=30$ | $30f_0$ | 若 $f_0=1/d$，则 $30f_0=1/30d$ | 月循环检测 |

**谐波结构的变化**：
- 谐波消失：某个周期性模式被中断（如周循环消失 = 工作-生活边界模糊）
- 谐波出现：新的周期性模式建立（如项目周期成为主导）
- 谐波重构：多个谐波的相对强度变化（如周循环增强，月循环减弱）

### 3.3 相位漂移（Phase Drift）

$$\Delta\phi(t) = \phi(t) - \phi(t - \Delta t)$$

其中 $\phi(t)$ 是瞬时相位：

$$\phi(t) = \arg\left[ W_x(a_{f_0}, t) \right]$$

**物理意义**：
- 同样的事件，发生的时间提前了还是滞后了
- **焦虑时加速**：相位提前（同样的事件，比预期更早发生）
- **抑郁时减速**：相位滞后（同样的事件，比预期更晚发生）
- **时差恢复**：相位从偏移逐渐回到正常

**映射到 Trit-Core `Phase`**：
- 直接对应 `Phase` 类型，范围 [0,1]
- 相位漂移速度 → 状态变化率的代理指标

### 3.4 频谱重构（Spectral Reconfiguration）

**检测**：Kullback-Leibler 散度

$$D_{KL}(P_{\text{new}} \| P_{\text{old}}) = \sum_f P_{\text{new}}(f) \log \frac{P_{\text{new}}(f)}{P_{\text{old}}(f)}$$

**物理意义**：
- 新频率成分的出现或旧频率成分的消失
- $D_{KL} > \theta$（阈值，典型值 0.5）→ 频谱重构发生

**认知意义**：
- 新关注域的出现（如开始关注健康，运动频率成为新谐波）
- 旧习惯的改变（如停止社交，社交频率的谐波消失）
- 新压力源的介入（如项目上线，项目周期成为新谐波）

**映射到 Trit-Core `MetaInterrupt`**：
- 触发 `SpectralReconfiguration` 类型
- 附带新旧频谱对比

### 3.5 跨信号相位同步（Cross-Signal Phase Synchronization）

**定义**：两个信号 $x$ 和 $y$ 的相位差的方差

$$\sigma_{\Delta\phi}^2 = \text{Var}\left[ \arg(W_x(a, t)) - \arg(W_y(a, t)) \right]$$

**同步指数**：

$$S_{xy} = e^{-\sigma_{\Delta\phi}^2}$$

- $S_{xy} = 1$：完全同步（相位差恒定）
- $S_{xy} = 0$：完全异步（相位差随机）

**认知意义**：
- 生理-行为同步高：身体状态和行为一致（健康）
- 生理-行为同步低：身体状态和行为矛盾（如身体疲惫但行为活跃，可能是透支）
- 社交-情绪同步高：社交活动和情绪状态一致
- 社交-情绪同步低：社交活动和情绪状态矛盾（如社交活跃但情绪低落，可能是表演）

**映射到 Trit-Core `Phase`**：
- 同步指数 → 系统整体协调性的代理指标

## 四、离散实现

### 4.1 离散小波变换（DWT）

对于实际计算，使用 DWT 而非 CWT：

$$W_x[j, k] = \sum_n x[n] \cdot \psi_{j,k}[n]$$

其中：
- $j$：分解层级（对应频率范围）
- $k$：时间平移
- $\psi_{j,k}[n]$：离散化的小波基函数

**分解层级与频率范围**：

| 层级 $j$ | 频率范围（以 $f_s$ 为采样频率） | 对应认知尺度 |
|----------|----------------------------|------------|
| $j=1$ | $f_s/4 - f_s/2$ | 分钟级（瞬时波动） |
| $j=2$ | $f_s/8 - f_s/4$ | 小时级（半日节律） |
| $j=3$ | $f_s/16 - f_s/8$ | 日级（昼夜节律） |
| $j=4$ | $f_s/32 - f_s/16$ | 周级（工作周期） |
| $j=5$ | $f_s/64 - f_s/32$ | 月级（项目周期） |
| $j=6$ | $f_s/128 - f_s/64$ | 年级（人生阶段） |

### 4.2 算法复杂度

- DWT：$O(N)$，线性时间（因为使用快速算法）
- CWT：$O(N \log N)$，因为对数尺度采样
- 对于 Aurora 的实时分析，使用 DWT 的滑动窗口变体

## 五、与 Trit-Core 的接口

### 5.1 小波输出 → Trit-Core 输入

```rust
pub struct SensorSignal {
    pub timestamp: DateTime<Utc>,
    pub source: SignalSource,        // 生理/行为/社交/环境
    pub feature: WaveletFeature,     // 基频/谐波/相位/频谱/同步
    pub value: f64,                   // 原始值
    pub phase: Phase,                 // 归一化到 [0,1]
    pub frame: Frame,                 // 对应的参考系
    pub confidence: f64,              // 信噪比（小波功率 / 噪声功率）
}
```

### 5.2 映射规则

| 小波特征 | Trit-Core 映射 |
|----------|---------------|
| 基频稳定性 | `Phase` 接近 0.5 = 稳定，偏离 0.5 = 漂移 |
| 谐波完整性 | 谐波数量 / 预期谐波数量 → `Phase` |
| 相位漂移速度 | 漂移速度归一化 → `Phase`（0=无漂移，1=极速漂移） |
| 频谱重构强度 | $D_{KL}$ 归一化 → `Phase`（0=无重构，1=完全重构） |
| 跨信号同步 | $S_{xy}$ 直接 → `Phase`（0=完全异步，1=完全同步） |

## 六、附录：母小波选择指南

| 母小波 | 特点 | 适用场景 |
|--------|------|----------|
| Morlet | 平衡时间-频率分辨率 | 一般认知节律分析（推荐） |
| Daubechies | 紧支撑，正交 | 需要快速计算、信号压缩 |
| Symlet | 近似对称 | 需要相位信息准确 |
| Coiflet | 高阶消失矩 | 需要平滑近似 |
| Haar | 最简单，不连续 | 需要检测突变边缘 |

**Aurora 默认使用 Morlet（CWT）和 Daubechies-4（DWT）。**
