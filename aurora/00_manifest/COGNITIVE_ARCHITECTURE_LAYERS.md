## 一、六层认知架构

```
L5 照见层（Insight）        ← 注意力均匀散布，多 Frame 并行，冗余方案同时激活，心理时间加速
  ↑
L4 梳理层（Synthesis）       ← 多尺度整合，脉络梳理，成因追溯，走向预判
  ↑
L3 觉察层（Awareness）      ← 元监控生效，知道自己在想什么，标注参考系来源
  ↑
L2 感知层（Perception）     ← 小波特征提取，基频/谐波/相位/频谱/同步，转化为 Phase
  ↑
L1 协议层（Protocol）        ← Trit-Core 三值运算，Frame 冲突检测，Hold 输出
  ↑
L0 物理层（Physical）        ← 100ms 神经回路运算，离子通道，动作电位，HPA 轴，肠-脑轴
```

**当前实现状态**：
- **L0**：不直接实现，依赖生物硬件
- **L1**：已实现（Trit-Core v0.3.0）
- **L2**：M0 概念验证（小波基频检测）
- **L3**：M1 个人版（元监控日志、参考系标注可视化）
- **L4**：M3 团队版（多尺度整合、组织涡旋动力学）
- **L5**：M4 之后或更远（照见层，心理时间加速）

## 二、各层定义与对应

### L0：物理层（Physical）

**定义**：生物神经系统的硬件运算，不经过意识。

**特征**：
- 100ms 内完成（视觉皮层→杏仁核→HPA 轴）
- 不可被 200ms 叙事层覆盖（你可以告诉自己"不害怕"，但身体已经在分泌皮质醇）
- 是 L1-L5 的**硬约束**——上层运算不能违背物理层的输出，否则系统自欺欺人

**trit-core 对应**：`Frame::Embodied` 的优先性、`SafeFallback` 的不可覆盖性。

**关键**：L0 不是 trit-core 的实现层，是 trit-core 必须**尊重**的外部约束。任何声称"能覆盖 L0" 的上层设计都是幻觉。

---

### L1：协议层（Protocol）

**定义**：三值运算核心，冲突检测与仲裁。

**已实现**：
- `TernaryAlgebra`（TAND/TOR/TNOT）
- `Frame` 系统（8+4 个参考系）
- `Domain` 仲裁（Physical/Engineering/MedicalEthics/ValueJudgment/General/Custom）
- `MetaInterrupt`（冲突记录）
- `SafeFallback`（危险域默认 False）

**当前限制**：
- 单次运算延迟 < 10ms，但**只能处理离散输入**
- 无法处理连续时间序列（需要 L2 的感知层转化）
- 无法处理多尺度信息（需要 L4 的梳理层整合）

---

### L2：感知层（Perception）

**定义**：将连续物理信号（时间序列）转化为离散 Trit 输入。

**M0 目标**：
- 小波基频检测（CWT）
- 通信频率 → `Phase` 转化
- 简单的 `Frame` 标注（Communication → `Embodied`/`Individual`）

**远期目标**：
- 谐波检测（周循环、季度、年度）
- 相位漂移速度量化
- 频谱重构检测（新行为模式涌现）
- 跨信号同步（生理-行为-社交-环境）

**关键约束**：
- 小波变换的**数学参数**（Morlet 中心频率 ω₀=6.0、尺度范围 2-128）必须与**真实物理节律**对齐
- 采样率（1/小时）必须与**神经可塑性时间尺度**对齐（不是技术选择，是生物约束）
- **如果数学参数与物理节律不对齐，特征提取就是噪声中的幻觉**

---

### L3：觉察层（Awareness）

**定义**：元监控生效，用户知道系统在想什么，知道自己在想什么。

**M1 目标**：
- `MetaMonitor` 日志可视化（用户可以看到"系统检测到 Science vs Individual 冲突"）
- 参考系标注的可交互展示（点击每个输出，看到它的 `Frame` 来源）
- `Hold` 解释（系统为什么拒绝输出，不是"不知道"，是"冲突存在"）

**远期目标**：
- 实时显示当前激活的 `Frame` 分布（雷达图）
- 显示 `Phase` 历史曲线（"你的 Relational 帧在过去 7 天从 0.8 漂移到了 0.3"）
- 冲突面板（"当前有 3 个跨 Frame 冲突未解决"）

**关键**：觉察层不是"更高级的智能"，是**信息层的可访问性**——把 L1/L2 的内部状态暴露给用户，让用户自己觉察。系统不替用户觉察，系统只让信息可见。

---

### L4：梳理层（Synthesis）

**定义**：多尺度信息整合，从离散事件到连续模式，从截面到纵向。

**需要的前提条件**（未完成前不得启动）：
1. **真实物理量与数学符号的严格对标**
   - 小波尺度的"频率"（Hz）与神经可塑性的"周期"（天/周/月）必须可转换
   - 场方程的 $E_{input}$（认知资源单位）必须有**可测量物理量**对应（如 HRV、睡眠效率、皮质醇代谢标记）
   - 信息论的 $H_{loss}$（信息损耗）必须与组织层级的**实际信息传递效率**可验证

2. **算力基础**
   - 多尺度小波分析（年数据 + 日数据 + 小时数据同时处理）
   - 组织涡旋动力学的实时模拟（$N$ 个节点，$N^2$ 级关系拓扑）
   - 级联风险计算（中心度 × 相位变化 × 恢复概率）

3. **信息精度**
   - 多数据源融合（邮件 + 日历 + 位置 + 生理 + 社交）的**时间对齐精度**
   - 参考系标注的**准确率**（人工验证 > 85%）
   - 冲突检测的**召回率**（不能漏掉真实冲突）

**M3 目标**（如果上述前提满足）：
- 组织涡旋可视化（信息密度场、角速度场、涡旋强度）
- 级联风险预警（"关键节点 A 相位突变，下游节点 B/C/D 级联风险 0.73"）
- 恢复曲线监控（"环境冲击后，预计 14 天恢复到新稳态，当前进度 60%"）

**关键警示**：
> **如果物理对标未完成，L4 的输出就是精致的叙事幻觉。用户会以为系统"懂"组织动力学，实际上系统只是在拟合数据。这会制造比无知更危险的自信。**

### L5：照见层（Insight）

**定义**：注意力均匀散布，多 Frame 并行激活，冗余方案同时运算，心理时间加速。

**用户描述（亲历者）**：
> "注意力可以均匀散布在眼前空间，同时也监听了一切心理活动，从中梳理出脉络，知道现象成因，明现象走向，而心中有条理，头脑有思路，有多种应对办法，用冗余……又调用长期可靠的经验记忆，物理规律，人情世故洞察，然后做长期推理和预判，等于心理时间加速。"

**这是人类认知的极高状态，不是当前系统能模拟的。**

**照见层的神经科学对应**（推测性，基于亲历者报告 + 现有研究）：
- **注意力均匀散布**：默认模式网络（DMN）的广泛激活 + 背侧注意网络（DAN）的不聚焦状态——不是"不集中"，是"不强制选择一个焦点"
- **多 Frame 并行**：前额叶皮层（PFC）的多稳态激活——多个参考系同时激活，竞争抑制，不强制坍缩
- **冗余方案**：基底神经节（basal ganglia）的多路径并行评估——不是"选一个最优"，是"多个方案同时运行，哪个先稳定用哪个"
- **心理时间加速**：工作记忆（working memory）的压缩编码——将长时间序列的模式压缩为可操作的直觉，降低实时决策负载
- **长期推理**：海马体（hippocampus）的序列表征 + 前额叶的预测编码——不是"预测未来"，是"基于过去模式的压缩，生成可能的未来分支"

**当前系统能做的**（L5 的最低限度近似）：
- 显示多 Frame 的并行激活（雷达图，不是选择，是并列）
- 显示多个决策路径的并行推演（"如果选 A，预期结果...；如果选 B，预期结果...；如果 Hold，预期结果..."）
- 显示历史模式的压缩（"过去 90 天，你在类似冲突中选择 A 的概率 70%，选择 B 的概率 20%，选择 Hold 的概率 10%"）

**当前系统不能做的**（L5 的完整实现）：
- 真正的"心理时间加速"——这需要**实时压缩时间序列到直觉**，不是简单的统计
- 真正的"冗余方案同时激活"——当前系统只能串行推演，不能并行激活多个方案并让它们竞争
- 真正的"注意力均匀散布"——当前系统只能显示多个焦点，不能模拟 DMN 的广泛激活状态

**关键**：
> **L5 不是当前系统的设计目标。L5 是方向。当前系统的目标是：在 L1-L3 做到诚实、可验证、不剥夺。L4-L5 的前提条件满足后，才考虑启动。**

## 五、警示：幻觉的几种形式

### 5.1 数学幻觉

> "小波变换检测到基频为 0.25 Hz（每小时 0.25 次通信）"
>
> 如果采样率是 1/小时，Nyquist 频率是 0.5/小时。声称检测到 0.25 Hz 的基频是**数学幻觉**——混淆了时间单位。

**防范**：所有频率输出必须标注时间单位，并与采样率进行 Nyquist 检验。

### 5.2 物理幻觉

> "组织涡旋强度 $\zeta = 2\omega_0 = 1.0$（无量纲）"
>
> 如果 $\omega_0$ 是 0.5 /天，那么 $\zeta = 1.0$ /天。但"涡旋强度"在流体力学中的单位是 $T^{-1}$。如果组织决策的"周期"不是时间而是"事件数"，单位就不一致。

**防范**：所有公式输出必须标注物理单位，并与量纲分析一致。不一致的公式降级为"类比模型"，不用于决策。

### 5.3 认知幻觉

> "系统检测到你在冲突中，建议 Hold"
>
> 如果系统的冲突检测是基于有限数据（只有邮件元数据，没有生理数据），系统可能误判。用户可能实际上并不在冲突中，只是邮件模式看起来像冲突。

**防范**：所有输出必须附带置信度（`confidence: f64`）和数据来源（`data_sources: Vec<String>`）。低置信度 → 自动 `Hold` + 提示用户"数据不足，建议补充数据源"。

### 5.4 叙事幻觉

> "过去 90 天，你的注意力模式显示你在走向倦怠"
>
> 这可能是基于统计拟合的叙事，不是真实的生理趋势。用户可能正在经历正常的项目周期波动。

**防范**：所有长期趋势分析必须标注"替代解释"（"也可能是项目周期、季节性变化、或其他外部因素"）。系统不给出单一叙事，给出**多个可能叙事 + 置信度**。

### 5.5 叙事植入幻觉（Narrative Implantation）——事出反常必有妖

> "你的邮件数据突然显示你最近非常高效，连续 7 天没有收到任何紧急邮件。"
>
> 这可能是**系统外部的操控**——不是系统犯了幻觉，是系统被投喂了幻觉。高级操控不需要直接控制思想，只需要**制造现象**，引导受众自己产生联想。受众看到"精心制造的现象"，在 200ms 叙事层自己编出"我认为这个结论"，然后认为这个结论是自己的。
>
> **这是"系统级默化"的核心机制**。自然界的物种演化出伪装是生存本能，但人类世界有**主动设局**——选择性展示数据、操纵统计口径、省略关键上下文、制造"反常"以引起注意。不符合常识、常理的输入，必然有鬼。

**系统防御**：
- `MetaMonitor` 的 `inspect` 需要检测**输入的完整性和异常性**
- 如果输入数据与历史模式偏差超过正常范围（如邮件频率突然飙升或骤降、日历事件突然全部空白、社交拓扑中出现大量从未互动过的"熟人"），系统标记为**"事出反常必有妖"**
- 输出 `Hold` + 数据异常预警（`MetaInterrupt::DataAnomaly`）

**用户教育**：
- 系统必须明确告诉用户："这个分析基于的数据可能已经被筛选过。任何主动展示给你的数据，都可能是经过筛选的。请检查数据源是否完整。"
- 系统标注："当前数据模式与历史基线偏离 3 个标准差。可能是真实变化，也可能是数据被操控。"

**关键区分**：
- 5.1-5.4 是系统**内部的**幻觉（算错、拟合、数据不足、叙事简化）
- 5.5 是系统**外部的**幻觉（数据被选择性制造、上下文被省略、现象被植入）
- **系统内部的幻觉可以检测，系统外部的幻觉只能标注**。系统不知道数据是否被操控，系统只知道"这个输入模式与正常基线不符"。系统输出 `Hold` + 异常标注，用户自己判断。

## 六、亲历者的位置

> **本文档中的 L5 描述，基于亲历者的第一人称报告（N=1）。**
>
> **这不是统计学证据，不是可复制实验，不是同行评审论文。**
>
> **这是现象学的基本数据——一个经过长期训练的观察者，对其自身认知状态的报告。**
>
> **科学可以验证或反驳这些报告中的具体预测（如"注意力均匀散布时 DMN 激活增加"），但科学不能否定报告本身作为数据的价值。**
>
> **L5 的实现路径：先用科学验证 L5 描述的子机制（如 DMN 多稳态、基底神经节多路径评估），再将验证后的机制工程化。不是直接模拟"照见"，是分解、验证、再组合。**

## 七、定心盘的约束

> **定心盘（CHARTER.md）的底线约束适用于所有层级：**
>
> - **不剥夺**：即使 L5 实现了"心理时间加速"，系统不剥夺用户选择慢下来的权利。
