# 人文关键词索引 — 科学化定义与说明

**Version**: 0.3.0
**Status**: Active

---

## 前言

本文档从 Trit-Core 项目的人文/哲学论述中提取关键概念，并用**科学方法**加以定义、标注和说明。所谓“科学化”，是指每个关键词的定义都尽量满足以下条件：

1. **可观察性**：定义指向可被观察、测量或操作化的现象，而非纯粹思辨实体。
2. **可区分性**：定义明确区分该概念与邻近概念，避免语义混叠。
3. **可验证性**：定义蕴含可检验的推论，至少原则上可以通过实验、临床观察或工程实践加以反驳或支持。
4. **条件性**：承认任何定义都是“在特定尺度、目的和约束下有效”的临时约定，而非绝对真理。

本文档的核心立场与项目一致：

> 现代科学主流范式追求普适性定律、公理化体系和可重复定理，这是人类认知的伟大成就；但它**无法覆盖每一位人类个体的全部实情**，也无法替代个体在具身状态和环境动态中的第一人称判断。身心在分析时可以分裂考察，但在实际起作用时是一个与环境和时间深度耦合的连续系统。因此，追问意识起源，可能需要改变旧有的、以“孤立观察者”为核心的探索方法。

## 关键词索引

### 1. 科学主流范式 / Mainstream Scientific Paradigm

**定义**
以数学化、形式化、可重复实验为特征的认知方法，其目标是发现跨情境普适的定律、公理和定理，并通过同行评审、预测能力和工程应用获得暂时性确认。

**为何如此定义**
科学不是“真理的占有者”，而是“可证伪假设的筛选机制”。将其定义为“主流范式”而非“真理本身”，是为了保留其历史性、可修正性和边界条件。该范式的力量在于**降低方差**——在大量样本和受控条件下提取稳定模式；其局限也正在于此：当样本量为 1（单个生命个体）且情境不可重复时，普适定律的预测力可能急剧下降。

**在 Trit-Core 中的体现**
`Frame::Science` 表示“基于当前可重复证据的暂时确定”。它不被等同于 `Frame::Consensus` 或 `Frame::Individual`，以避免把统计规律升格为个体真理。

**参见**
- `Frame::Science`
- `统计共识 ≠ 真知`
- `可验证性`
- [`DAO-SCIENCE-REFERENCES.md`](DAO-SCIENCE-REFERENCES.md) §1（L0–L7 认知频谱框架）

### 2. 普适性 / Universality

**定义**
一个命题或定律在不同个体、情境、时间和空间中保持形式不变的能力。在数学和物理学中达到最高程度（如守恒律）；在生物学、心理学和人文领域，普适性通常只存在于高度抽象的约束层面，具体表现具有极强的情境依赖性。

**为何如此定义**
普适性不是一个二值属性（是/否），而是一个**程度属性**，取决于观察尺度、变量选择和控制条件。将普适性视为“程度”而非“本质”，可以避免在人文领域错误地套用硬科学标准，也可以避免在硬科学领域错误地引入相对主义。

**在 Trit-Core 中的体现**
核心代数（TAND/TOR/TNOT）是普适的；但 `ResolutionPolicy` 是域特定的——`Physical` 与 `MedicalEthics` 对同一冲突给出不同仲裁，这是对“普适性有边界”的工程承认。

**参见**
- `Domain`
- `不可通约性`
- `临时约定`

### 3. 公理、定理、真理 / Axiom, Theorem, Truth

**定义**
- **公理**：在一个形式系统中被预设为真的命题，作为推导起点，本身不在该系统内被证明。
- **定理**：从公理出发，通过演绎规则得到的命题；其“真”是相对于公理系统和推理规则而言的。
- **真理**：一个更广泛的哲学概念，指命题与实在之间的某种对应关系；在科学实践中，通常以“尚未被证伪且能成功预测”的操作化方式使用。

**为何如此定义**
这一定义区分了三个常被混用的概念：公理是**约定**，定理是**推导结果**，真理是**理想目标**。科学中的“定理”之所以可靠，是因为它被限制在明确的形式系统内；一旦离开该系统（例如应用于不可形式化的个体经验），定理的适用性就需要重新论证。

**在 Trit-Core 中的体现**
`TritValue::True` 和 `TritValue::False` 不是“真理”，而是“在特定参考系和证据条件下的可计算判定”。`Hold` 和 `Unknown` 则分别标记“条件不足”和“超出系统范围”。

**参见**
- `TritValue`
- `可验证性`
- `诚实 / 不知`

### 4. 统计共识 / Statistical Consensus

**定义**
大量主体在特定时间、文化和测量方式下对某一命题表现出的频率优势。它是**分布的中心趋势**，不是对该命题真实性的证明。

**为何如此定义**
统计共识的可观察性很好（可以计数），但它与“真”之间没有逻辑必然性。历史上，地心说曾是统计共识； today 的某些健康误区也可能是统计共识。将该概念与“真知”分离，是为了防止把“很多人相信”误当成“这件事为真”。

**在 Trit-Core 中的体现**
`Frame::Consensus` 明确标记“这是统计或群体偏好”，与 `Frame::Science`（经验证据）和 `Frame::Individual`（个体实情）区分。`ResolutionPolicy` 不会自动把 Consensus 升级为 Science。

**参见**
- `Frame::Consensus`
- `真知 / 实践真知`
- `群体认知髓鞘化`

### 5. 真知 / 实践真知 / Validated Knowledge

**定义**
经过可重复实践检验、能够成功预测或干预现象、且尚未被证伪的暂时性认知。真知不是“绝对正确”，而是“在当前证据和实践范围内最可靠”。

**为何如此定义**
“真知”如果定义为“与实在完全一致的命题”，则无法被任何有限主体声称拥有。将其操作化为“经得起实践检验”，既保留了科学的现实主义追求，又避免了独断论。真知的暂时性意味着它随时准备被新证据修正。

**在 Trit-Core 中的体现**
`Frame::Science` 表示“基于当前可重复证据的暂时确定”。`Hold` 表示“证据不足或不可通约，本系统拒绝判定”。

**参见**
- `Frame::Science`
- `可验证性`
- `统计共识 ≠ 真知`

### 6. 个体实情 / 第一人称实情 / Individual Reality

**定义**
特定人类主体在不可完全外化的具身状态、历史、关系网络和第一人称体验中呈现出的实在。它可能与统计平均、科学定律或群体共识不一致，但在该主体的生命情境中具有最终裁决权。

**为何如此定义**
个体实情不是一个反科学概念，而是一个**尺度概念**。科学方法在群体层面有效，但无法穷尽单个主体的全部变量。第一人称体验（如疼痛、意义感、恐惧）具有不可还原的主观维度，任何第三方测量都会损失信息。因此，在科学无法覆盖的真空地带，个体实情的权重应当更高。

**在 Trit-Core 中的体现**
`Frame::Individual` 表示个人上下文/个人事实。`MedicalEthics` 域中 Individual 帧具有优先级；`ValueJudgment` 域永远返回 `Hold`，因为算法不应替代个体做价值判断。

**参见**
- `Frame::Individual`
- `第一人称检验`
- `身心二元 / 身心一体`
- [`DAO-SCIENCE-REFERENCES.md`](DAO-SCIENCE-REFERENCES.md) §3（第一人称认识论）

### 7. 第一人称检验 / First-Person Verification

**定义**
以主体自身的直接体验作为某一命题（尤其涉及痛苦、意义、偏好）的最终检验标准。它不是替代客观测量，而是在客观测量失效或不足时的补充证据。

**为何如此定义**
某些事实（如“我是否痛苦”、“这个选择对我是否有意义”）无法被外部仪器完全捕获。第一人称检验不是“主观随意”，而是对这些特殊事实的**最适认识论工具**。它的可靠性可以通过主体间的沟通、长期行为一致性和神经生理关联来间接验证。

**在 Trit-Core 中的体现**
`MedicalEthics` 域的 Preserve(Individual) 规则，是对“患者第一人称体验在医疗决策中具有最终权重”的工程实现。

**参见**
- `个体实情`
- `可验证性`
- `MedicalEthics`
- [`DAO-SCIENCE-REFERENCES.md`](DAO-SCIENCE-REFERENCES.md) §3（第一人称认识论）

### 8. 身心二元 / 身心一体 / Mind-Body Duality vs. Embodiment

**定义**
- **分析性身心二元**：在研究方法上，可以暂时将认知过程（心）与生理过程（身）分开考察，以便分别建立模型。
- **功能性身心一体**：在实际起作用时，心智是大脑、神经系统、内分泌系统、免疫系统、躯体感觉和环境互动的动态整体。分析性二元是工具，功能性一体是事实。

**为何如此定义**
身心二元（如笛卡尔传统）如果作为**本体论**断言，会陷入困境；但如果作为**方法论**工具，它是有效的。认知科学、神经科学和现象学都表明：情绪、决策和“直觉” deeply 依赖身体状态（如心率、肠道菌群、肌肉张力、疲劳程度）。因此，分析时可以分，但解释实际行为时必须合。

**在 Trit-Core 中的体现**
`Phase`（连续相位）可被视为“身体倾向”的形式化：它表达“我尚未决定，但强烈倾向某一方”，保留了身体感知与理性判断之间的张力，而不是用强制二值化消灭它。

**参见**
- `Phase`
- `躯体状态`
- `环境动态`

### 9. 躯体状态 / Somatic State

**定义**
主体身体的实时生理配置，包括但不限于神经系统活动、内分泌水平、内脏感觉、肌肉张力、疲劳程度、疼痛信号和自主神经平衡。它是心智决策的底层输入之一。

**为何如此定义**
躯体状态不是“认知偏差”的来源，而是**信息来源**。神经科学中的内感受（interoception）研究表明，身体信号对决策、风险评估和自我意识至关重要。将躯体状态定义为“底层输入”，是为了把它从“需要被抑制的噪声”重新定位为“需要被倾听的数据”。

**在 Trit-Core 中的体现**
`Phase` 和 `Hold` 可以编码“身体尚未准备好做决定”的状态。`ValueJudgment` 域的 Hold 也适用于“身体/情绪信号与理性分析冲突”的情境。

**参见**
- `Phase`
- `身心二元 / 身心一体`
- `环境动态`

### 10. 环境动态 / Environmental Dynamics

**定义**
主体所处物理、社会和符号环境的实时变化，包括他人行为、文化语境、资源可用性、风险分布和信息流。环境不是静态背景，而是与主体持续互动的动态系统。

**为何如此定义**
传统认知科学常把“环境”当作可控制的背景变量；但真实决策中，环境是**共同演化**的。同一主体在不同环境中可能做出不同决策，这不是“非理性”，而是适应性。将环境动态纳入定义，可以解释为何普适模型在个体层面经常失效。

**在 Trit-Core 中的体现**
`Frame::Consensus` 可被视为环境动态的符号层面（社会期望、文化规范）。跨 Frame 冲突（如 Individual vs Consensus）本质上是个体与环境动态之间的张力。

**参见**
- `Frame::Consensus`
- `Frame::Individual`
- `认知生态学`

### 11. 意识起源 / Origin of Consciousness

**定义**
主观体验（qualia）如何从物质过程中产生的问题，即“困难问题”。它不询问意识的功能或神经相关物（NCC），而询问**为何会有任何事物被体验**。

**为何如此定义**
这一定义采用了 Chalmers 的“困难问题”框架，以区分意识的**功能问题**（做什么）和**存在 problem**（为何存在）。Trit-Core 的立场是：如果旧有方法（孤立观察大脑、寻找单一意识中心）长期无法解决困难问题，那么可能需要转向**关系性、具身性和过程性**的方法——把意识视为身心-环境-时间互动网络中的涌现属性，而非大脑内部的孤立产物。

**在 Trit-Core 中的体现**
`Frame::Absolute` 标记“不可知或不可观测”的命题，其中可能包括意识的终极起源。系统对这些命题输出 `Hold`，而不是假装可以用科学框架判定。

**参见**
- `Frame::Absolute`
- `身心二元 / 身心一体`
- `不可知 / 不可判定`

### 12. 悬置 / Hold / Suspension

**定义**
在信息不足、跨域冲突或不可通约情境下，系统有意识地不做二值判定，同时保留未来在条件变化时重新判定的可能性。

**为何如此定义**
悬置不是“失败”、“未解析”或“中性平均”，而是一个**积极的认知操作**。在直觉主义逻辑中，它对应“未证明”；在工程上，它是 IEC 61508 安全原则的补充（“我不知道”在某些域意味着“不要做”）。将 Hold 定义为一等计算状态，是对自然心智能力的工程映射。

**在 Trit-Core 中的体现**
`TritValue::Hold` 是可计算状态，与 `Unknown`（不可计算）严格区分。跨 Frame 操作默认返回 `Hold` + `MetaInterrupt`。

**参见**
- `TritValue::Hold`
- `TritValue::Unknown`
- `不可通约性`
- [`DAO-SCIENCE-REFERENCES.md`](DAO-SCIENCE-REFERENCES.md) §2（知止不殆与 AI 停止标准）

### 13. 不可通约性 / Incommensurability

**定义**
两个或多个价值、量纲或参考系之间不存在共同度量单位，因此无法通过加权平均或多数决合理合并的状态。

**为何如此定义**
不可通约性不是“暂时没发现换算方法”，而是“根本不存在换算方法”。例如，“个人自由”与“公共安全”不能放在同一个效用函数里求和，因为分母不存在。承认这一点，是避免把政治选择伪装成数学优化的前提。

**在 Trit-Core 中的体现**
`ValueJudgment` 域永远返回 `Hold`，因为价值判断不可通约。`SafeFallback` 在危险域把 Hold 强制转 False，是对“不可通约不等于无决策”的工程处理。

**参见**
- `ValueJudgment`
- `Hold`
- `对齐的元问题`

### 14. 群体认知髓鞘化 / Cognitive Myelination

**定义**
借用神经科学中的髓鞘化概念，指群体中某些认知模式因反复被 AI 输出、媒体内容和社交互动激活而变得更加自动化、更难改变的过程。

**为何如此定义**
髓鞘化是神经可塑性的具体机制：重复激活的神经元通路被髓鞘包裹，传导速度加快，但也更难抑制。当数亿人每天阅读由同一组模型权重生成的文本时，这些文本中的隐含框架（“什么是合理的”、“什么是正常的”）可能通过反复激活而被加固。这不是比喻，而是对大规模信息环境的神经可塑性推论。

**在 Trit-Core 中的体现**
`MetaInterrupt` 和不可篡改的决策日志，是防止群体认知髓鞘化的审计机制。系统必须能回答：“为什么选择了这个输出而不是 Hold？”

**参见**
- `MetaInterrupt`
- `统计共识`
- `认知生态学`
- [`DAO-SCIENCE-REFERENCES.md`](DAO-SCIENCE-REFERENCES.md) §1（L0–L7 认知频谱框架）

### 15. 认知生态学 / Cognitive Ecology

**定义**
把人类认知视为一个生态系统，其中包含多样性的信息源、框架、主体和媒介。健康的认知生态需要多样性、可审计性和负反馈机制，单一权重集合的爆炸性增长可能导致认知失衡。

**为何如此定义**
生态学的核心洞见是：单一物种的优势往往降低系统韧性。将这一洞见应用于信息环境，可以解释为何“所有决策都被迫坍缩为二元输出”会压缩认知多样性，从而削弱集体智慧。

**在 Trit-Core 中的体现**
`Hold` 状态保留了多重视角共存的可能性，而不是强制合并为一个平均值。`Frame` 系统本身就是对认知多样性的形式化保护。

**参见**
- `Frame`
- `Hold`
- `群体认知髓鞘化`

### 16. 可验证性 / Verifiability

**定义**
一个命题或系统输出能够被独立主体通过观察、实验或复现来检验其是否符合预期的性质。它是科学方法的伦理底线，也是工程系统的问责基础。

**为何如此定义**
可验证性把知识从“权威宣称”转变为“公共检验”。在 Trit-Core 中，可验证性不仅适用于科学结论，也适用于系统行为：每个决策都必须能被复现和审计。

**在 Trit-Core 中的体现**
所有场景保存为 JSON，可通过 `cargo test` 和 `cargo run --bin trit-sandbox` 复现。`MetaInterrupt` 日志提供可审计的冲突记录。

**参见**
- `MetaInterrupt`
- `第一人称检验`
- `真知 / 实践真知`

### 17. 二值思维 / 三值思维 / Binary vs. Ternary Thinking

**定义**
- **二值思维**：将世界切分为两个互斥类别（是/非、对/错、安全/危险）的认知策略。它是节能的速度优化，但会损失中间状态和不可比较性信息。
- **三值思维**：在 True/False 之外保留一个明确的 Hold（悬置）状态，用于标记冲突、不确定或不可通约。

**为何如此定义**
二值思维不是“错误”，而是**工具性简化**。它在生存紧急情境中有优势，但在复杂人文决策中会导致范畴错误。三值思维不是否定二值思维，而是在其边界处补充一个“拒绝判断”的选项。

**在 Trit-Core 中的体现**
`TritValue` 枚举明确包含 True、Hold、False 三个可计算状态，以及 Unknown 元标记。跨 Frame 操作返回 Hold。

**参见**
- `TritValue`
- `Hold`
- `悬置`

### 18. 临时约定 / Provisional Convention

**定义**
在特定域、特定约束和特定目的下被证明有效，但不声称具有跨情境绝对有效性的规则或定论。

**为何如此定义**
所有工程规则、伦理原则甚至科学定律都是临时约定。它们的价值不在于“永恒正确”，而在于“在已知条件下有效且可修正”。将规则明确定性为临时约定，可以防止其僵化成为教条。

**在 Trit-Core 中的体现**
`Physical` 域强制 Commit Science、`ValueJudgment` 域永远 Hold、`SafeFallback` 在危险域强制 False——这些都是域特定约定，不是绝对真理。

**参见**
- `Domain`
- `SafeFallback`
- `普适性`

### 19. 诚实 / 不知 / Epistemic Honesty

**定义**
一个认知系统明确承认自己知识边界的能力：当面对不可判定、不可通约或超出训练分布的问题时，选择说“我不知道”而不是编造一个看似合理的答案。

**为何如此定义**
在 AI 对齐语境中，最危险的行为不是“犯错”，而是“假装知道”。 epistemic honesty 把“承认无知”从缺陷重新定义为美德。它是可审计系统的前提。

**在 Trit-Core 中的体现**
`TritValue::Hold` 是“我知道这个问题，但基于当前信息拒绝判定”；`TritValue::Unknown` 是“这个输入超出我的认知范围”。两者都是诚实的不同形式。

**参见**
- `TritValue::Hold`
- `TritValue::Unknown`
- `悬置`

### 20. 参考系 / Frame / Reference Frame

**定义**
决策判断所依赖的隐含背景规则、价值优先级和证据标准。同一事实在不同参考系中具有不同的意义和行动含义。

**为何如此定义**
参考系概念来自物理学（如惯性参考系、洛伦兹变换），强调“在不同参考系中测量的量不能直接比较，需要变换”。将这一概念引入决策论，可以形式化“跨域冲突”：来自不同参考系的判断不能直接取平均。

**在 Trit-Core 中的体现**
`Frame` 枚举（Science、Individual、Consensus、Absolute、Meta）是参考系的形式化。同 Frame 操作走热路径，跨 Frame 操作返回 Hold + MetaInterrupt。

**参见**
- `Frame`
- `不可通约性`
- `Domain`
- [`DAO-SCIENCE-REFERENCES.md`](DAO-SCIENCE-REFERENCES.md) §1（L0–L7 认知频谱框架）

### 21. 对齐的元问题 / Alignment Meta-Problem

**定义**
不是“如何让 AI 与人类价值观对齐”，而是“当人类自己的价值观互相冲突时，AI 应该怎么做？”这个问题无法通过优化单一目标函数解决，因为选择目标函数本身就是一种价值观行为。

**为何如此定义**
传统 AI 对齐把“人类价值观”当作一个可平均的静态集合；但实际上，人类价值观是情境依赖、动态演化且内部冲突的。对齐的元问题把冲突本身置于中心，要求 AI 检测冲突、明确标记，并交还决策权。

**在 Trit-Core 中的体现**
`ResolutionPolicy` 不追求“最优输出”，而是根据 Domain 规则进行域特定仲裁。`ValueJudgment` 域永远 Hold，把决策权交还人类。

**参见**
- `ResolutionPolicy`
- `ValueJudgment`
- `不可通约性`

### 22. 不可知 / 不可判定 / Undecidable

**定义**
在当前认知框架、证据条件或测量能力下，无法被判定为真或假的命题或状态。它可能是暂时的（等待新证据），也可能是原则上的（超出可知边界）。

**为何如此定义**
不可知不是“虚无主义”，而是对可知边界的诚实标记。哥德尔不完备定理表明，任何足够强大的形式系统都存在不可判定命题；Trit-Core 把这一数学事实工程化。

**在 Trit-Core 中的体现**
`Frame::Absolute` 标记不可知/不可观测的命题，任何 Absolute 帧的 TritWord 必须永远 Hold。

**参见**
- `Frame::Absolute`
- `意识起源`
- `哥德尔视角`
- [`DAO-SCIENCE-REFERENCES.md`](DAO-SCIENCE-REFERENCES.md) §1（L0–L7 认知频谱框架）

## 核心论点总结

1. **科学范式有其边界**：普适定律在群体层面有效，但无法穷尽个体实情的全部变量。
2. **身心在分析中可分，在作用中一体**：理性分析可以暂时抽象身体，但实际决策必须考虑躯体状态和环境动态。
3. **意识起源可能需要新方法**：如果孤立观察大脑长期无法解决困难问题，应转向关系性、具身性和过程性视角。
4. **Hold 是诚实而非缺陷**：在不可通约、信息不足或跨域冲突时，系统应主动悬置判断。
5. **所有定论都是临时约定**：规则的有效性依赖于域、约束和目的，必须随时准备修正。
