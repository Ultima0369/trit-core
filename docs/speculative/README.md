# Speculative / Vision Documents

This directory contains design documents for features that are purely
aspirational — no implementation exists in the codebase. They are kept here
for reference and future inspiration, not as current specifications.

## Contents

- `MONOLITHIC_BOT_SPEC.md` — VoxelWorld embodied intelligence system (4-layer
  cognitive architecture). Zero code exists. Kept as a design reference.
- `EMBODIMENT_INDEX.md` — Original index for the embodiment documentation
  subtree. Many referenced files do not exist.

These were moved from `docs/explanation/embodiment/` during the 2026-07-05
CTO audit to reduce confusion between documented vision and implemented reality.
